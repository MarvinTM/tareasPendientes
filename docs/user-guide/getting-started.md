# Getting Started with Tareas Pendientes

## Overview

Tareas Pendientes is a family task management application designed to help households organize, assign, and track tasks collaboratively. It features a Kanban-style board with real-time updates, making it easy for family members to stay synchronized.

## Accessing the Application

### First-Time Login

1. Navigate to the application URL in your web browser
2. Click the **"Sign in with Google"** button
3. Select your Google account and authorize the application
4. **Wait for admin approval** - New users require approval from an administrator before accessing the task board

### Pending Approval Status

If you see a "Pending Approval" message after logging in:
- An administrator has been notified of your registration
- You will gain full access once an admin approves your account
- Contact your family administrator if approval is taking too long

## Main Interface

Once approved, you'll see the main task board with three columns:

| Column | Spanish Name | Description |
|--------|--------------|-------------|
| **New** | Nueva | Tasks waiting to be started |
| **In Progress** | En Progreso | Tasks currently being worked on |
| **Completed** | Completada | Finished tasks |

## Creating Your First Task

1. Click the **"+" button** or **"Nueva Tarea"** (New Task)
2. Fill in the task details:
   - **Title** (required): Brief description of the task
   - **Description** (optional): Additional details
   - **Category** (required): Select from available categories (e.g., Cleaning, Shopping)
   - **Size/Difficulty**:
     - **S (Peque√±a)**: Small task (~1 point)
     - **M (Mediana)**: Medium task (~2 points)
     - **L (Grande)**: Large task (~3 points)
   - **Assigned To** (optional): Family member responsible
3. Click **"Create"** or **"Crear"**

## Managing Tasks

### Moving Tasks Between Columns

- **Drag and Drop**: Click and hold a task card, then drag it to another column
- Tasks automatically update for all family members in real-time

### Editing a Task

1. Click on any task card to open the edit dialog
2. Modify the fields as needed
3. Click **"Save"** or **"Guardar"**

### Deleting a Task

1. Click on the task to open the edit dialog
2. Click the **"Delete"** or **"Eliminar"** button
3. Confirm the deletion

## Viewing Task History

To see the history of changes for a specific task:

1. Click on a task card
2. Select **"View History"** or **"Ver Historial"**
3. You'll see a chronological list of all changes (status updates, assignments, edits)

## Scoreboard

Track family contributions via the scoreboard:

1. Navigate to **Scoreboard** from the menu
2. View scores by time period:
   - **This Week**: Current week's performance
   - **This Month**: Current month's totals
   - **This Year**: Year-to-date
   - **All Time**: Complete history

Points are awarded based on task size:
- Small (S): 1 point
- Medium (M): 2 points
- Large (L): 3 points

## Email Notifications

When a task is assigned to you:
- You'll receive an email notification (if configured)
- The email includes the task title, description, and who assigned it
- This helps you stay informed even when not actively using the app

## Tips for Effective Task Management

1. **Be specific with titles**: "Clean kitchen counters" is better than "Clean"
2. **Use categories**: Helps organize and filter tasks
3. **Assign appropriately**: Distribute tasks fairly among family members
4. **Update task status promptly**: Keeps everyone informed
5. **Check the scoreboard**: Motivates participation and shows who's contributing

## Responsive Design

The application adapts to different screen sizes:
- **Desktop**: Full three-column view with all task details visible
- **Tablet**: Optimized column layout
- **Mobile**: Compact cards that expand on tap, columns may be reduced

## Keyboard Shortcuts

Currently, the application is primarily mouse/touch operated. All actions are accessible through the graphical interface.

## Troubleshooting

### Can't see the task board?
- Check if you're logged in
- Verify your account has been approved by an administrator

### Tasks not updating?
- Check your internet connection
- Refresh the page to re-establish the WebSocket connection

### Email notifications not arriving?
- Check your spam/junk folder
- Confirm email notifications are configured (ask your administrator)

## Need Help?

Contact your family administrator for:
- Account approval
- Category management
- Technical issues
